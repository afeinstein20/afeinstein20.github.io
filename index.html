Research Interests

Broadly, I am interested in observational exoplanet and stellar astrophysics. In the future, I hope to focus more on planet formation around low-mass stars, with an interest in binary systems.

Current Project -- Identifying Planet Candidates with TESS

Currently, at the University of Chicago, I am developing a set of Python tools for exploring the Transiting Exoplanet Survey Satellite (TESS) Full-Frame Images (FFIs). This package will not only create lightcurves for all sources in the TESS Input Catalog for Tmag < 16 that will be hosted on the MAST servers, but will also be able to create custom lightcurves for sources in Gaia. Users will be able to create custom target pixel files, lightcurves, and useful figures and videos for visualzing their data, such as the example below. You can follow my progress on GitHub! The documentation for this pipeline can be found here along with examples on how you can use ELLIE. The GIF below demonstrates one of the features of ELLIE, using an exlipsing binary found in simulated TESS data (End-To-End 6; hosted on MAST). Users will be able to create time-series movies of their sources as well as track the associated light curve with the pixel data.

Former Project -- Gravitational Lensing

Example of a strongly lensed galaxy, the dragon, caused by the presence of the foreground galaxy cluster, Abell370.

I've had the fortune to work on several projects throughout my undergraduate career. First and foremost, my undergraduate research conducted at Tufts University focued on calculatin gravitational lensing factors and associated errors for sources located in the Hubble Frontier Fields campaign. I estimated the magnification factor of thousands of galaxies, using the strong lensing equation, as part of the HFF-DeepSpace program (P.I. Marchesini). Our collaboration, as well as my personal contribution of the lensing catalogs, is publicly available and can be accessed here. The publication associated with this work can be found on in the Astrophysical Journal Supplement Series. Below is a strong lensing event found using the galaxy cluster Abell370. To the left, a red-green-blue image of the cluster with a zoomed in area around a lensed background galaxy (lovingly named the dragon). To the right, a segmentation map of the lensing cluster and the associated magnification values of the galaxy (yellow = small lensing factor; black = large lensing factor).
